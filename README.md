# Pakistan Wonders - HTML/CSS/JS Version

This is a pure HTML, CSS, and JavaScript version of the Pakistan Wonders website, showcasing the incredible wonders of Pakistan.

## Project Structure

```
html_version/
├── index.html      # Main homepage
├── 404.html        # Error page
├── styles.css      # All CSS styles
└── main.js         # JavaScript functionality
```

## Features

- ✅ Fully responsive design
- ✅ Smooth scrolling navigation
- ✅ Interactive wonder cards
- ✅ Dynamic featured wonder display
- ✅ Beautiful gradient backgrounds
- ✅ Animated scroll effects
- ✅ No dependencies - pure vanilla HTML/CSS/JS

## How to Use

1. **View the Website:**
   - Open `index.html` in any modern web browser
   - No server or build process required!

2. **Customize:**
   - Edit `index.html` for content changes
   - Modify `styles.css` for styling adjustments
   - Update `main.js` for interactive features

## Sections

1. **Hero Section** - Eye-catching introduction with call-to-action
2. **Stats Section** - Key statistics about Pakistan
3. **Wonders Section** - Interactive showcase of 6 iconic Pakistani wonders:
   - Hunza Valley
   - Mohenjo-daro
   - Fairy Meadows
   - Badshahi Mosque
   - Chitral Kalash Valleys
   - Swat Valley
4. **Creator Section** - Information about the website creator
5. **CTA Section** - Call-to-action for exploration
6. **Air University Section** - Featured institution
7. **Footer** - Navigation and copyright information
